# Welcome to the vagSpaceX wiki!

## To start the project

* `git clone https://github.com/Barbe-Bleue/vagSpaceX.git`
* `cd vagSpaceX/`
* `npm i`

Start Angular server

`ng serve -o`

go to [localhost:42000](localhost:42000)
